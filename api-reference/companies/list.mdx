<RequestExample>

```bash Filter by IPO status and industry
curl --request POST \
  --url https://www.tryfundable.ai/api/v1/companies \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{"company":{"ipo_status":["private"],"industries":["artificial-intelligence"]},"latest_deal":{"financing_types":[{"type":"SERIES_A"},{"type":"SERIES_B"}]},"page_size":10}'
```

```bash AI-powered semantic search
curl --request POST \
  --url https://www.tryfundable.ai/api/v1/companies \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{"company":{"search_query":"AI healthcare diagnostics startups","min_relevance":0.5},"page_size":5}'
```

```bash Look up by domain
curl --request POST \
  --url https://www.tryfundable.ai/api/v1/companies \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{"identifiers":{"domains":["stripe.com","airbnb.com","notion.so"]}}'
```

</RequestExample>
